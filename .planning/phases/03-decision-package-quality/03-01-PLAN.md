## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| prioritized shortlist → 主决策包生成器 | 非 UI 的结构化事实输入进入管理层产物 |

| 生成器 → shared state | 文件写入从脚本越过到共享资产目录 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | T | `scripts/generate_decision_package.py` backlink schema | mitigate | 仅允许使用固定 `upstream_paths[]` 与 `idea_id/judgment_id` 字段生成回链，拒绝自由文本证据引用，降低 evidence-link tampering。 |
| T-03-02 | E | 三个新生成器的写路径 | mitigate | 在骨架阶段即固定写入目录常量，只允许决策包、执行包、董事会简报、trace 目录，禁止写受保护共享状态，防止 generator overreach。 |
| T-03-03 | I | 执行包/董事会简报输入边界 | mitigate | 派生器脚本中不声明 triage 或 role artifact 输入常量，只接受主包路径，降低 derived artifact inconsistency。 |
