| Boundary | Description |
|----------|-------------|
| analysis loop wrapper → 生成器链 | shell 顺序调用决定哪些产物会在 daily loop 默认生成 |

| smoke test → 发布门禁 | 自动验证决定坏链路是否能继续流入后续阶段 |

| T-03-10 | E | `orchestration/cron/commands.sh` | mitigate | 保持 thin wrapper，不把新写共享状态逻辑放进 cron helper，避免 wrapper overreach into protected shared state。 |
