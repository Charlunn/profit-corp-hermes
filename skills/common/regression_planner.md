# Skill: regression-planner（中立版）

- 名称：regression-planner
- 版本：v0.2
- 来源：Google SRE/SWE 测试可靠性思想（重写）

## 适用场景
- 功能变更后回归范围设计

## 非适用场景
- 无代码行为变化

## 输入要求
- 改动清单
- 影响模块
- 历史缺陷热点

## 执行步骤
1. 基于改动影响选取回归范围。
2. 建立回归清单：核心路径、边界路径、历史易错路径。
3. 划分执行层级：PR 快速回归/合并后扩展回归。
4. 输出阻断标准与放行条件。

## 输出契约
- 回归范围说明
- 回归用例清单
- 放行/阻断条件

## DoD
- [ ] 覆盖关键链路与高风险邻接区域
- [ ] 放行条件明确
- [ ] 回归结果可追踪

## 风险与回滚
- 风险：回归范围过窄导致线上回归
- 回滚：扩大回归范围并冻结发布
