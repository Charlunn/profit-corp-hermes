# Skill: bugfix-safety（中立版）

- 名称：bugfix-safety
- 版本：v0.2
- 来源：google/eng-practices, pre-commit/pre-commit（重写）

## 适用场景
- 线上/测试缺陷修复
- 需要严格控制回归风险

## 非适用场景
- 需求开发

## 输入要求
- 问题现象与期望行为
- 最小复现步骤
- 影响范围

## 执行步骤
1. 先复现并固化证据。
2. 定位根因并区分类型（逻辑/状态/依赖）。
3. 最小修复并避免搭车重构。
4. 回归验证主路径与相邻高风险路径。
5. 输出问题闭环记录。

## 输出契约
- 复现证据
- 根因说明
- 修复摘要
- 回归证据
- 回滚步骤

## DoD
- [ ] 修复前后对比证据完整
- [ ] 回归通过且无新增分层违规
- [ ] 回滚可执行

## 风险与回滚
- 风险：猜修导致问题迁移
- 回滚：按变更单元回滚，并恢复已变更配置
