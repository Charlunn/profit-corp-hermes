# Skill: information-hierarchy-optimizer（中立版）

- 名称：information-hierarchy-optimizer
- 版本：v0.2
- 来源：GOV.UK, USWDS, NN/g（重写）

## 适用场景
- 页面信息过载、用户找不到重点

## 非适用场景
- 极简单一操作页面

## 输入要求
- 页面内容块列表
- 用户核心任务

## 执行步骤
1. 按用户任务排序信息优先级。
2. 校验标题层级与视觉层级一致性。
3. 减少首屏非关键信息干扰。
4. 输出重排方案与理由。

## 输出契约
- 信息层级评估
- 重排建议
- 风险提示

## DoD
- [ ] 首屏聚焦核心任务
- [ ] 标题层级与阅读路径一致
- [ ] 冗余信息有折叠或后置策略

## 风险与回滚
- 风险：过度压缩导致信息缺失
- 回滚：保留关键补充信息在次级层
