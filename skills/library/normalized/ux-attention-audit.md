# Skill: ux-attention-audit（中立版）

- 名称：ux-attention-audit
- 版本：v0.2
- 来源：NN/g, Material/Carbon/Primer（重写）

## 适用场景
- 页面交互设计评审
- 关键转化页面优化

## 非适用场景
- 纯后端改动

## 输入要求
- 页面目标与主任务
- 当前页面结构（区域/按钮/文案）

## 执行步骤
1. 识别主任务与主 CTA。
2. 检查注意力竞争（多个高权重按钮、视觉噪声）。
3. 调整视觉层级（主/次/辅助操作）。
4. 给出可执行改进建议。

## 输出契约
- 主任务清晰度评估
- 注意力冲突列表
- 优化建议与优先级

## DoD
- [ ] 每个页面主任务可一眼识别
- [ ] 主次操作层级清晰
- [ ] 改进建议可落地

## 风险与回滚
- 风险：过度简化导致功能入口隐藏
- 回滚：保留必要次级入口，分层展示
