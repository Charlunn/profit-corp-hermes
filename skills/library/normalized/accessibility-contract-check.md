# Skill: accessibility-contract-check（中立版）

- 名称：accessibility-contract-check
- 版本：v0.2
- 来源：W3C WCAG/APG, Primer（重写）

## 适用场景
- 前端交互上线前可访问性校验

## 非适用场景
- 无用户界面的纯后台作业

## 输入要求
- 页面结构与组件清单
- 关键用户任务路径

## 执行步骤
1. 检查语义结构（标题、地标、label）。
2. 检查键盘可达与焦点顺序。
3. 检查颜色对比与非颜色提示。
4. 检查组件 ARIA 与可读文本一致性。

## 输出契约
- 可访问性检查结果
- 风险等级
- 修复建议

## DoD
- [ ] 关键任务可键盘完成
- [ ] 语义结构完整
- [ ] 对比度与提示机制达标

## 风险与回滚
- 风险：视觉可用但交互不可达
- 回滚：恢复语义优先实现并补充无障碍标注
