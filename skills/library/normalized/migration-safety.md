# Skill: migration-safety（中立版）

- 名称：migration-safety
- 版本：v0.2
- 来源：工程实践资料（重写）

## 适用场景
- 数据库 schema 变更
- 数据迁移/回填任务

## 非适用场景
- 无数据结构变更的纯业务逻辑调整

## 输入要求
- 迁移目标
- 数据规模与回填策略
- 回滚策略

## 执行步骤
1. 评估迁移影响（锁、时长、并发写）。
2. 采用可回滚方案（分阶段迁移优先）。
3. 设计前后兼容窗口（读写双路径如有必要）。
4. 执行前演练与执行后验证。

## 输出契约
- 迁移方案摘要
- 风险评估
- 验证结果
- 回滚步骤

## DoD
- [ ] 风险评估覆盖并发与锁影响
- [ ] 有可执行回滚方案
- [ ] 执行后验证通过

## 风险与回滚
- 风险：长事务/锁表影响线上
- 回滚：按预案回退 schema 与数据变更步骤
