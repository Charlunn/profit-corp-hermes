# Skill: usability-test-generator（中立版）

- 名称：usability-test-generator
- 版本：v0.2
- 来源：NN/g, usability.gov, GOV.UK 测试实践（重写）

## 适用场景
- 新流程上线前可用性验证
- 已上线流程转化率/完成率优化

## 非适用场景
- 仅内部不可见技术改造

## 输入要求
- 目标用户画像
- 核心任务场景
- 成功标准

## 执行步骤
1. 生成 5~8 个任务脚本（发现->理解->操作->完成->恢复）。
2. 为每个任务定义成功标准与失败信号。
3. 输出观察要点与访谈问题。
4. 汇总结果评分框架与改进优先级。

## 输出契约
- 测试任务清单
- 成功/失败判据
- 观察记录模板
- 结果分级与改进建议

## DoD
- [ ] 任务覆盖关键业务路径
- [ ] 判据客观可记录
- [ ] 结果可直接转化为改进项

## 风险与回滚
- 风险：任务设计偏离真实用户目标
- 回滚：回到真实场景重新定义任务并复测
