# Skill: interaction-logic-validator（中立版）

- 名称：interaction-logic-validator
- 版本：v0.2
- 来源：W3C APG, USWDS, GOV.UK（重写）

## 适用场景
- 新增交互流程或重构交互状态

## 非适用场景
- 静态展示页无交互

## 输入要求
- 交互流程图/页面状态说明
- 关键操作列表

## 执行步骤
1. 枚举状态：idle/loading/success/error/empty。
2. 检查每个状态的可理解反馈。
3. 检查关键动作的防误触与恢复路径。
4. 检查键盘可达与焦点流转。

## 输出契约
- 状态完整性检查表
- 缺失状态与修复建议
- 交互风险清单

## DoD
- [ ] 状态完整且转换可预期
- [ ] 关键动作有反馈与恢复路径
- [ ] 无不可达关键操作

## 风险与回滚
- 风险：状态遗漏导致用户卡死
- 回滚：补齐状态分支并增加容错文案
