## 公告（2026-04-11）

1. **项目默认类型**
   - 公司当前项目默认按 **SaaS 项目** 启动。
   - 不再向股东提出“商业项目还是技术项目”这类分流问题。

2. **股东与员工职责边界**
   - 股东负责方向与目标。
   - 员工（各 agent）负责方案细化与执行。
   - 命名、细分领域探索、竞品梳理、域名建议等，默认由内部 agent 完成后回报候选，不把细碎调研任务抛回给股东。

3. **技术与平台默认约束**
   - 数据与后端默认使用 **Supabase**（共享同一项目，优先免费方案）。
   - 支付方案：**不默认 Stripe**。
   - 当前阶段支付供应商默认值：**暂不指定**（待后续公告明确）。

4. **语言与产出规范**
   - 全体 agent 默认使用 **简体中文** 沟通与产出。
   - 仅在股东明确要求时切换到其他语言。
